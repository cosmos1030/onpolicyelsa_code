ELSA + On-Policy Knowledge Distillation

ELSA м•Ңкі лҰ¬мҰҳ

мөңм Ғнҷ” л¬ём ң:

$$\min_{w,\,z}\; \mathcal{L}(w) \quad \text{s.t.}\quad w = z,\quad z \in \mathcal{S}_s$$
Algorithm 1: ELSA with On-Policy KD

Input: Dense model $\pi_T$ (teacher), prompt dataset $\mathcal{D}$, sparsity $s$, penalty $\lambda$

Initialize: $w \leftarrow \theta_\text{dense},\quad z \leftarrow \text{Proj}_{\mathcal{S}_s}(w),\quad u \leftarrow 0$

for $t = 1, \ldots, T$ do

1. Sample prompt $x \sim \mathcal{D}$

2. Generate $y \sim \pi_w(\cdot \mid x)$   (student, no gradient)

3. Compute $\mathcal{L}(w)$   вҶҗ On-Policy KD loss (м•„лһҳ м°ёмЎ°)

4. w-update:

$$g \leftarrow \nabla_w \mathcal{L}(w) + \lambda(w - z + u), \qquad w \leftarrow w - \eta\,\text{Adam}(g)$$

5. if $t \bmod \texttt{interval} = 0$:

$$z \leftarrow \text{Proj}_{\mathcal{S}_s}(w + u), \qquad u \leftarrow u + (w - z)$$

end for

return $\text{Proj}_{\mathcal{S}_s}(w)$

NTP:

$$\mathcal{L}_\text{NTP}(w) = -\,\mathbb{E}_{y^* \sim \mathcal{D}}\!\left[\sum_{t} \log \pi_w\!\left(y^*_t \mid y^*_{<t},\, x\right)\right]$$

KD (offline): teacher л¶„нҸ¬лҘј нғҖк№ғмңјлЎң, кі м •лҗң $y^*$ мң„м—җм„ң кі„мӮ°.

$$\mathcal{L}_\text{KD-off}(w) = \mathbb{E}_{y^* \sim \mathcal{D}}\!\left[\sum_{t} D\!\left(\pi_T(\cdot \mid y^*_{<t}, x)\;\Big\|\;\pi_w(\cdot \mid y^*_{<t}, x)\right)\right]$$

On-Policy KD: studentк°Җ мғқм„ұн•ң $y$лҘј мӮ¬мҡ© вҶ’ covariate shift м—ҶмқҢ.

$$\mathcal{L}_\text{KD-on}(w) = \mathbb{E}_{y \sim \pi_w(\cdot|x)}\!\left[\sum_{t} D\!\left(\pi_w(\cdot \mid y_{<t}, x)\;\Big\|\;\pi_T(\cdot \mid y_{<t}, x)\right)\right]$$
